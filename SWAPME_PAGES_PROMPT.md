## 1. Live-тикер курсов (шапка сайта)

Бегущая строка прямо под хедером (или встроенная в хедер), которая непрерывно скроллится слева направо.

### Реализация
- Бесконечная CSS-анимация `marquee` (translateX от 0 до -50%)
- Дублированный контент для seamless loop
- Монеты с иконками: BTC ₿, ETH Ξ, USDT ₮, TON ◆, SOL
- Каждая монета показывает: иконку + название + цену в USD + изменение за 24ч (зелёный/красный)
- Высота: 36-40px, фон чуть темнее хедера (#111)
- При hover — анимация останавливается (animation-play-state: paused)

### Данные
- Первый этап: захардкоженные цены, обновлять вручную
- Второй этап: CoinGecko API (бесплатный, без ключа)
  ```
  https://api.coingecko.com/api/v3/simple/price?ids=bitcoin,ethereum,tether,the-open-network&vs_currencies=usd&include_24hr_change=true
  ```

### Визуал
```
◈ BTC $104,250 ▲+2.3%  ·  ◈ ETH $3,890 ▼-0.8%  ·  ◈ USDT $1.00 ─ 0.0%  ·  ◈ TON $6.42 ▲+5.1%
```
- Зелёный (#00C853) для роста, красный (#FF3D00) для падения
- Иконки монет маленькие (16px), цветные
- Разделитель — тонкая точка с glow

## 7. Форма заявки на обмен

Встроенная форма прямо на сайте (не только ссылка на Telegram).

### Поля
- Имя / Name
- Telegram @username (обязательно)
- Направление обмена: выпадающий (привязан к калькулятору)
- Сумма
- Комментарий (опционально)

### Поведение
- При отправке: данные уходят в Telegram через бот API (webhook)
- На первом этапе: просто `window.open('https://t.me/swapme_bot?text=...')` с pre-filled сообщением
- Pre-fill из калькулятора: если клиент использовал калькулятор, данные автоматически подставляются в форму
- Success state: "Заявка отправлена! Мы свяжемся с вами в Telegram в течение 5 минут"

### Визуал
- Glassmorphism карточка
- Animated border (gradient rotating)
- Валидация полей в реальном времени
- Кнопка с ripple effect

## 8. Telegram Chat Widget

Плавающая кнопка в правом нижнем углу (как Intercom/Crisp).

### Реализация
- Круглая кнопка 56px с иконкой Telegram (или маскотом-лисёнком)
- Оранжевый фон, белая иконка
- Пульсирующий glow эффект для привлечения внимания
- При hover: scale(1.1)
- При клике: открывает мини-окно с двумя опциями:
  1. "Написать в Telegram" → `https://t.me/swapme_bot`
  2. "Оставить заявку на сайте" → скролл к форме
- Tooltip при первом визите: "Нужна помощь? Напишите нам!" (исчезает через 5 сек)
- На мобиле: кнопка меньше (48px), без tooltip

### Анимация появления
- Появляется через 3 секунды после загрузки страницы
- Slide-up + fade-in
- Bounce эффект один раз для привлечения внимания

## 9. Роутинг (Multi-page SPA)

### Навигация
Так как это React SPA, используем hash-router или простой state-based роутинг:

```
/           → Главная (всё из Prompt #1)
/about      → О нас / Команда
/blog       → Блог (лента постов)
/blog/:slug → Пост
/partners   → Партнёрская программа
/history    → История обменов
```

### Хедер навигация
- Лого (→ главная)
- Обмен (→ калькулятор на главной, scroll)
- О нас
- Блог
- Партнёрам
- [RU/EN переключатель]
- CTA кнопка "Обменять" (→ калькулятор или Telegram)

### Переходы между страницами
- Fade transition (opacity 0→1, 300ms)
- Scroll to top при переходе
- Active state в навигации — оранжевый underline

## 10. Дополнительные визуальные фишки

### Preloader (загрузка сайта)
- Логотип SwapMe по центру
- Две стрелки анимированно "обмениваются" (белая ↔ оранжевая)
- Progress bar снизу
- Фон: #1A1A1A
- Исчезает через 2 сек (или по загрузке)

### Easter Egg
- При клике на маскота 5 раз: лисёнок подмигивает (CSS animation)
- Или: секретный промокод появляется ("FOXSWAP" — скидка на комиссию)

### Scroll Progress Bar
- Тонкая оранжевая линия вверху страницы
- Показывает прогресс скролла (0-100%)
- Высота: 3px, position: fixed

### Back to Top
- Кнопка-стрелка вверх, появляется после скролла 500px
- Оранжевый круг с белой стрелкой
- Smooth scroll to top

### Cookie Consent
- Минимальный баннер внизу
- "Мы используем cookies" + кнопки "Принять" / "Отклонить"
- Glassmorphism стиль
- Исчезает после выбора
